What is the difference between Margin, Border and Padding?

 
Margin is the area outside of a defined border, for e.g if you go to the Youtube homepage and continuously scroll upwards there is a black, blank area above the search bar. With margin properties you can set the margin for each side of an element.
 
Border is the outline or frame surrounding the content on a webpage. This can be seen as the grey box surrounding the Youtube search bar. Border properties in CSS allow for customization where you can change the style, width, and color of an element's border.
 
Padding is to create space around content (what people see in the browser), inside of defined borders. Again going back to the Youtube example, this is the difference between the border and content inside. With CSS you have full control, where you can use padding-properties to specify the padding of each side (e.g top, right, bottom, left) to your liking.
